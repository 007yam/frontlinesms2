# Sent Archive

---

_The Sent archive will contain messages that have been archived from the Folder. The layout and features of the Inbox archive will be explained in this help page._

---

### [Sent Archive][1]

The Sent archive contains all messages that have been archived from the [Sent Folder][2]. 

![alt text][17]

#### [Sent Archive header][3]

The Sent Archive Header includes options for [Quick Message][4] and [Export][5].

#### [Sent Archive Message List][6]

The Message list here functions like any other message list elsewhere in the program. Multiples messages can be selected, starred or filtered. To learn more about how to do this, click [here][7]

#### [Sent Archive Message Details][8]

When a message is selected, the full text will be displayed here along with the ability to [reply][9], [forward][10], [unarchive][11], [delete][12] and [move][13].
