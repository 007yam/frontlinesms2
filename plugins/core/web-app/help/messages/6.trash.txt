# Using the Trash Folder

_The trash folder contains all the messages you have deleted. The messages are stored here where they can be restored or deleted permanently._

(Screenshot)

Please see [here][2]

### [Emptying the trash][3]

To empty the trash folder of all messages:

__NOTE:__ This will _permanently_ delete all messages in the trash folder

1.  From the 'Trash Actions..' dropdown menu (1), select "Empty Trash".

All messages in the trash folder will be permanently deleted.
