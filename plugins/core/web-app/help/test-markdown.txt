# Getting Around the Messages Tab

_The messages tab is shown by default whenever you start FrontlineSMS. From here you can easily see all your [incoming messages][1], [sent messages][2], [activities][2], and [folders][3]. You can also create [new activities][4] and [new folders][5] from this tab. You can also view the content of individual messages and manage your messages with actions like [delete][6], [reply][7], [archive][8] and [move][9]._

### Navigation tab

This bar will allow you access the various parts of the software: [Archive][10], [Contacts][11], [Status][12], [Search][13] and [Settings and Plugins][14]. <these will each link to the overview pages of each tab>

### [Message List][15]

This list displays all the messages you have received and that haven't been sorted into activities or folders.   
Messages that have not been read will be displayed in bold, while messages you have read will not.
Clicking on a single message displays its contents on the right hand side in the [Message Preview][16] area. The selected message also becomes highlighted.

### [Navigation Pane][17]     
- __[Inbox][18]:__ 
Clicking here will show you all your incoming messages that are not affiliated with an [activity][2] or [folder][3].
- __[Sent][19]:__
Clicking here will show you all the messages which have been successfully sent from FrontlineSMS. 
- __[Pending][20]:__
Clicking here will show you all the messages which are waiting to be sent or could not be sent.  
Messages that could not be sent will be indicated with an error icon (insert pic).   
Go to [Message History][14] for further information on why an error message has occurreds. 
- __[Trash][21]:__
Clicking here will show you all the messages that you have deleted. Any messages associated with an activity or folder will be found here and no longer associated to that activity.

__[Activities][2] and [Folders][3]:__  
- From here you can [create new activities][2] and navigate to existing ones which will be listed here.
- You can also [create new folders][23] and navigate to existing ones that are listed here.

### [Message Details][24]
When a message or multiple messages are selected in the message list, the message details will display further information about the message or messages.

When a single message is selected, the message details will display the full text of the message. You can also choose to [reply][7], [delete][6], [archive][8], [move][9] or [forward][25] the message.

When multiple messages are selected, a summary of how many messages have been selected will be shown in place of the message text. You can also choose to [reply][7], [delete][6], [archive][8], and [move][9] the messages.
